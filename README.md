# Shortcodes
## Detail
Can be used in two ways:
```html
{{< detail "Title" "Text" />}} <!-- For Inline -->

{{< detail "Title" >}}
Text here
Text here
Text here
{{</ detail >}} <!-- For Multiline -->

```

## Rawhtml

```html

{{< rawhtml >}}
	<div> Hi </div>
{{< /rawhtml >}}
```
